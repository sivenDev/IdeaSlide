关于缩略图生成

缩略图可以在保存时自动生成，有几种策略：

A

实时生成

每次编辑后立即更新缩略图，保证预览始终最新，但频繁操作时可能影响性能

B

延迟生成

保存文件时才生成缩略图，性能好但预览可能不是最新状态

C

混合策略

编辑时用内存缓存的预览，保存时才写入 thumbnails/，平衡性能和体验
